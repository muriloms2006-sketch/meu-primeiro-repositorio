As tags para formatação de texto são:

  • <b> - Texto em negrito
  • <strong> - Texto importante
  • <i> - Texto italico
  • <em> - Texto enfatizado
  • <mark> - Texto marcado
  • <small> - Texto menor
  • <del> - Texto excluído
  • <ins> - Texto inserido
  • <sub> - Texto subscrito
  • <sup> - Texto sobrescrito

A tag <pre> preserva a formatação do texto.

        Estudar na IFSC é legal

Listas

  • Lista desordenada: igual a essa que está sendo utilizada
  • Lista ordenada: invés de símbolos, são utilizados números
  • Lista de definição: não possui um símbolo. Ela possui termos e um texto curto sobre cada termo

A lista desordenada utiliza a marcação <ul>. Cada item utiliza a marcação <li>.

Exemplo:

Alimentos saúdaveis

  • Água
  • Carnes Brancas
  • Frutas, verduras, legumes

A lista ordenada utiliza a marcação <ol>. Os itens também utilizam <li>.

Exemplo:

Classificação dos jogos:

  1. Equipe Turing
  2. Equipe Berners-Lee
  3. Equipe Djkistra

Imagens

A marcação que inclui uma imagem na página é a img. O atributo src contém a URL da imagem, que pode ser absoluta ou relativa. A URL relativa segue as mesmas regras que as URLs de hyperlinks. Outro atributo que é obrigatório na imagem é o alt. Ele exibe um texto no lugar da imagem. Isso é util para navegadores que não exibem imagens e também para deficientes visuais.

Exemplo
  • <img src="" alt="" />
  • <img src="cachorro.gif" alt="Cachorro" />
  • <img src="fotos/viagem.jpg" alt="Viagem das férias" />

links <a>

Em HTML, um hiperlink é uma referência (um endereço) para um recurso da web. Para criar um hiperlink utilizamos a tag que irá definir um elemento clicável na página, e irá indicar o caminho do recurso disponibilizado por este.

Sintaxe Texto do link

<a href="url">Texto do link</a>

O endereço a qual o link irá levar é definido pelo atributo href. Importante ressaltar que o elemento clicável não necessariamente precisa ser um texto, podemos utilizar uma imagem, por exemplo.
